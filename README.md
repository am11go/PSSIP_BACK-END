ИНСТРУКЦИЯ ПО УСТАНОВКЕ НА ХОСТИНГ (ИЛИ ЛОКАЛЬНЫЙ СЕРВЕР)

ШАГ 1: Перенос файлов
Зайдите в панель управления хостингом (или используйте FTP-клиент, например FileZilla). Если используете локальный сервер (OpenServer/XAMPP), откройте папку домена (например, htdocs или domains/perfect-doors.local).
Загрузите в эту директорию папку assets/ и все файлы проекта: index.html, admin.php, catalog.php, db.php, login.php, save_order.php, send_mail.php и остальные скрипты.

ШАГ 2: Настройка базы данных
В панели управления хостингом найдите раздел "Базы данных MySQL" (или зайдите в phpMyAdmin).
Создайте новую базу данных с именем: perfect_doors
Создайте пользователя базы данных и назначьте ему пароль (сохраните эти данные для следующего шага). Для локального сервера этот пункт можно пропустить.
Зайдите в phpMyAdmin, выберите созданную базу данных perfect_doors и перейдите во вкладку "Импорт" (Import).
Нажмите "Выберите файл" и загрузите файл дампа perfect_doors.sql, который находится в папке вашего проекта. Нажмите "Вперед" (Импорт). В базе появятся таблицы: categories, products, orders, contacts и заполнятся тестовые товары.

ШАГ 3: Подключение сайта к базе данных
Найдите в корне сайта конфигурационный файл подключения — db.php
Откройте файл db.php для редактирования.
Укажите актуальные настройки подключения (те, что вы задали или выдал хостинг):
Хост (Host): обычно 'localhost'.
Имя БД (DB Name): perfect_doors
Пользователь (User): ваш_пользователь (или 'root' для локалки).
Пароль (Password): ваш_пароль (или пусто для локалки).
Сохраните изменения в файле.

ШАГ 4: Проверка работы сайта
Откройте адрес вашего сайта в браузере. Должна загрузиться главная страница (index.html) с каталогом и информацией о компании.
Проверьте работу форм обратной связи: нажмите "Вызвать замерщика" или "Отправить запрос", заполните данные и отправьте. Запрос должен обработаться скриптом send_mail.php (или сохраниться в БД).
Перейдите на страницу каталога (допишите в адресной строке /catalog.php), чтобы убедиться, что товары успешно подгружаются из базы данных.
Для проверки системы управления перейдите по адресу /login.php, введите логин и пароль для доступа к admin.php. Убедитесь, что там отображаются заказы и товары.
